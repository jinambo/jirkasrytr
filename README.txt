-------------------------------------------------------------
Vítej, toto je vysvětlení a návod pro tuto front-end šablonu.
-------------------------------------------------------------
HTML SOUBORY

index.html    - samozřejmě hlavní stránka
iflogin.html  - hlavní stránka, pokud je uživatel přihlášen
login.html    - slouží k registraci a přihlášení (pro smartphony je to jediná možnost loginu nebo registrace)
post.html     - slouží pro přidání příspěvků na fórum
diskuze.html  - příklad toho, jak budou vypadat jednotlivé diskuze
profile.html  - předloha pro uživatelský profil
settings.html - nastavení uživatelského profilu
-------------------------------------------------------------
Jak by mělo fungovat přidávání příspěvků?

V souboru post.html je možnost výběru tématu, kterého se daný příspěvek bude týkat.
Pokud uživatel vybere například možnost "grafika", jeho příspěvek to odešle do kategorie PGM.
